Setup & Integration Guide

  1. One-click start: just run oris-mania-utils.exe. It auto-detects TOSU, installs the overlays into tosu/static/, and remembers the paths for next time.
  2. Fallback setup: if auto-detect misses your TOSU or osu! Songs folder, pick them once in the manager. After that, the app keeps them saved.
  3. Enable Tosu In-Game Overlay: open http://127.0.0.1:24050/settings and verify that In-Game Overlay is toggled ON.
  4. Add overlays in osu!: launch osu!, open Tosu's menu with the default keybind, and add these overlays: msdconverter, ManiaKeystrokes, and HitCounter.
  5. Browser / OBS sources: if you want to use them in OBS or a browser, use these local URLs:
    MSD Radar: http://localhost:24050/msdconverter/
    Keystrokes: http://localhost:24050/ManiaKeystrokes/
    Hit Counter: http://localhost:24050/HitCounter/
    (Click any link above to copy and open in browser)
